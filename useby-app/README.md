# UseBy

UseBy is a neighbourhood inventory-to-action engine for dense communities. This app is the Vercel/Next.js front end and server runtime for the H0 hackathon build.

## Getting Started

Install dependencies and run the local app:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Verification

```bash
npm run lint
npm run typecheck
npm run test
npm run build
```

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Drizzle ORM
- AWS RDS Data API for Aurora PostgreSQL
- S3 private storage

See `docs/checkpoint-0-handoff.md` for the Checkpoint 1 starting context.
